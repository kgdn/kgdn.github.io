Let's get started! So I'll try make this my first, actual post. If anyone was there to see my original two posts, no you weren't. So, let's try this again.

Let's start with the burning question - why? Why did I make this website? Sheer boredom, I suppose. A test of my ability to write a blog, and to make a website. (I'm no front-end developer, but I'm pretty proud of what I made here today.) But hey, that's alright. I'm not going to be a front-end developer for a while, if ever.

I remember a good couple years ago I made a blog on that Neocities website, it's basically Geocities but... new, neo? Neo. And that website was bad, it was poorly designed and just looked terrible. Looking back at the HTML for that website made my skin crawl. But I guess I've improved since then, and that's probably a good thing since I'm a lot older now.

Funny thing is, this isn't the first time that I've made a GitHub website. I actually did it a couple of months ago as a part of my web development course (which god knows how I passed). But that was basically done for us in a way, the CSS and HTML were done for us... we just had to piece it all together and make a website. So, come a few months later and I gave it another go, and here we are! With an RSS feed and a comment section, as well. I'll probably add more stuff to this later, such as a search engine and post tags. But for now, I'm getting tired and frankly I would like to sleep. So, I'll just leave this here.
